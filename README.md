Challege corso Algoritmi avanzati per il calcolo ad alte prestazioni
Consegna:
- Usando la struttura dati Disjoin Set, scrivere un algoritmo che verifica se esiste un percorso tra ingresso e l'uscita in un labirinto (no diagonali). Il labirinto è memorizzato in un file.
- Scrivere un algoritmo che genera in maniera casuale un labirinto a due dimensioni dato:
• Il numero di righe e di colonne
• le coordinate del punto di ingresso
• le coordinate del punto di uscita
Il file generato deve avere lo stesso formato del punto precedente
